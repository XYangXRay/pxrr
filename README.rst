Installation
------------

To install the `pxrr` package in a local development environment, follow these steps:

1. Clone the repository:

.. code-block:: bash

    git clone https://github.com/nsls2/pxrr.git
    cd pxrr

2. Create the pixi environments (default and 'dev'):

.. code-block:: bash

    pixi install --all

3. That's all!
